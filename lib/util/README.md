# util/

Shared helper utilities used across the codebase.

## Files

- `auto_party.js`: Automated party invite/accept logic.
- `distance.js`: Distance and proximity helpers.
- `functions_data.js`: Data helpers for working with game constants/objects.
- `functions_game.js`: Gameplay helper utilities.
- `idle.js`: Simple idle counter for status/heartbeat.
- `logger.js`: Logging helpers with consistent formatting.
- `time.js`: Timing helpers (sleep, throttle, etc.).

## How it works

- Imported by most modules to keep logic small and consistent.
